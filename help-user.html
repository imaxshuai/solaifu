    Q1 :怎么注册搜来福帐号？

    您只需要按照以下步骤操作，就可以注册账号啦：
1. 点击搜来福首页右上方的“注册”。
2. 输入注册搜来福的注册信息，并点击获取手机校验码。
3. 获取校验码之后，在该页面内输入收到的校验码。
4. 即可注册成功。

    Q2 :怎么登录搜来福账户？

    您只需要按照以下步骤操作，就可以登录搜来福账户：
1. 点击搜来福首页右上角的【登录】按钮。
2. 输入当时注册的用户名、密码，点击【登录】，即可登录成功！

    Q3 :怎么更换绑定搜来福账号的手机号码？

    1. 进入搜来福首页，点击首页右上方“用户中心”。
2. 点击相关认证，再点击手机号那一列的“修改手机号”。
    3. 修改绑定手机，要验证以前手机号码。输入旧手机号码，获取校验码，填写校验码！点击去修改。
4. 验证新的手机信息，验证成功即修改成功！。

    Q4 :忘记密码和账号怎么办？

    1. 点击搜来福首页右上方的“登录”。
2. 页面跳转至登录页面后，点击页面上的“忘记密码”。
3. 按照页面提示，输入想要登录搜来福账号所绑定的手机号码，并点击【确定】。
4. 输入手机号,输入获取得到的校验码，点击【确定】。
5. 按照要求输入新的密码和确认密码即可修改成功！

    Q5 :为什么注册账户时提示手机号码已注册过？

    当您注册账号时提示您的用户名已经注册时， 则表示您的这个手机号码或用户名可能已经被您或其他用户注册过了，
您可以修改密码，重新“登录”。

    Q6 :搜来福帐号是否可以注销？

    目前不可以。
